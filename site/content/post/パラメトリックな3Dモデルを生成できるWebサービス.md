[Iteration3D Is Parametric Python In The Cloud](https://hackaday.com/2025/12/11/iteration3d-is-parametric-python-in-the-cloud/)から発見。画像もここから転載。

パラメトリックな3Dモデルを生成できるWebサービスであるIteration3Dの紹介です。

たくさんのパラメトリックな3Dモデルに対して、好きなようにパラメータを設定した3Dデータをダウンロードできます。
内部的にはCadQueryとbuild123dが動いており、STLデータを生成しています。

3Dプリンタで作りたい立体物のうちのいくらかは、このように素朴な形状の物なので、ゼロからモデリングするのではなく、既存のテンプレートにパラメータを適用するだけでデータを作ることができるのは嬉しいです。
